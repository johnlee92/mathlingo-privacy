어린이와 보호자를 위한 요약

2. 처리하는 정보, 목적과 보유 기간

구분정보목적보유 기간
계정과 인증 보호자 이메일, Supabase 계정 ID, 로그인 세션, 인증 요청 시각, IP 주소와 기기·앱·보안 로그 등 자동 생성 정보 6자리 로그인 코드 발송, 계정 식별, 로그인 유지, 부정 사용 방지, 계정 삭제 확인 계정 삭제 시 활성 계정·세션 원본을 삭제합니다. 프로젝트 데이터베이스의 인증 감사 로그 저장은 비활성화되어 있으며, 외부 보안 로그·복구 사본은 Supabase의 요금제·보안 정책에 따른 기간 동안 제한적으로 남을 수 있습니다.
서버 학습 정보 학습자 표시 이름·사용자 이름·아바타 글자와 색상·가입일, 선택 학년·코스·레슨·유닛과 진도, 별·XP·정확도·콤보·시도 횟수·완료 시각, 오답과 복습 기록, 젬·하트·연속 학습·프리즈·XP 부스트, 일일 활동·퀘스트·배지·리그, 온보딩 여부, 일일 목표, 동기화용 임의 기기 ID·변경 ID·리비전과 관련 시각 학습 상태 저장, 다른 기기와 동기화, 충돌·중복 저장 방지, 학습 기능 제공 계정 삭제 시까지
기기 내 오프라인 캐시 게스트·계정별 학습 스냅샷 전체(프로필 포함), 대기 중인 변경·리비전·임의 기기 ID·변경 ID·순번과 관련 시각 등 동기화 메타데이터, Keychain에 저장되는 로그인 세션 오프라인 학습, 앱 재실행과 재로그인 뒤 이어하기, 서버 동기화 충돌 방지 로그아웃 뒤에도 재로그인을 위해 계정별 캐시가 남을 수 있으며, 계정 삭제 또는 앱 삭제 시 정리됩니다.
기기 안에만 저장 진행 중인 레슨의 현재 문제·입력 상태, 리마인더·사운드·햅틱·동작 줄이기와 안내 닫기 설정 중단한 학습 이어하기와 기기별 사용 환경 제공 앱 데이터 초기화 또는 앱 삭제 시까지. 서버에는 동기화하지 않습니다.

3. 선택 동의와 만 14세 미만 아동

계정 연결과 서버 동기화는 선택 사항입니다. 동의하지 않으면 계정 연결과 다른 기기 동기화를 이용할 수 없지만, 게스트 모드로 이 기기에서 학습할 수 있습니다.

만 14세 미만 아동은 법정대리인이 개인정보 처리 내용을 확인하고 동의한 경우에만 계정을 연결해야 합니다. 앱은 연결 전에 보호자 동의 확인란 선택과 입력한 이메일의 인증 코드 확인을 요구합니다. 이 절차는 이메일 접근 권한을 확인하지만, 동의 여부·시각·처리방침 버전·아동과의 관계를 별도로 저장하거나 확인하지 않습니다. 법정대리인은 아래 문의처를 통해 아동의 개인정보 열람·정정·삭제·처리정지 및 동의 철회를 요청할 수 있습니다.

4. 처리위탁과 서비스 인프라

수탁자업무처리 위치와 기간
Supabase, Inc. 및 그 인프라 제공자 이메일 인증, 계정·데이터베이스·파일 저장, 보안·장애 대응 주 데이터베이스는 AWS 서울 리전(ap-northeast-2)에 위치합니다. 서비스 운영·지원 과정에서 Supabase 및 공개된 하위처리자가 국외에서 제한적으로 처리할 수 있으며, 계정 또는 위탁 계약 종료 시까지 처리합니다.
Google LLC (Gmail SMTP 활성화 시) 보호자 이메일로 일회용 인증 코드 발송 이메일 주소와 인증 메일 내용이 암호화된 통신으로 Google의 운영 인프라에서 발송 처리되며, Google의 서비스·보안 정책에 따른 기간 동안 처리될 수 있습니다.
GitHub, Inc. 개인정보 처리방침 웹페이지 호스팅과 보안 페이지 방문 시 IP 주소와 요청 로그가 GitHub의 운영 인프라에서 보안 목적으로 처리되며, GitHub의 서비스·보안 정책에 따른 기간 동안 보관될 수 있습니다.

시행일 현재 Gmail SMTP는 활성화되어 있지 않으며 Google LLC에 인증 메일 발송을 위탁하지 않습니다. 활성화할 때 이 문구와 처리 현황을 갱신합니다.

Supabase의 최신 처리 조건과 하위처리자 정보는 Supabase Data Processing Addendum에서 확인할 수 있습니다.

5. 제3자 제공과 광고

MathLingo 운영자는 이용자의 개인정보를 판매하지 않으며 맞춤형 광고에 사용하지 않습니다. 법령상 의무가 있거나 이용자가 별도로 동의한 경우를 제외하고, 위 처리위탁 목적 밖의 제3자에게 제공하지 않습니다.

  • 열람·정정·삭제·처리정지·동의 철회 요청은 ubergriden@gmail.com으로 접수할 수 있습니다. 요청자 본인 또는 정당한 법정대리인인지 확인할 수 있습니다.
